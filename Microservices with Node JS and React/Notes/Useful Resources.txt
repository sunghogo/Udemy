Section 25: [Appendix B] - Basics of Typescript:
    To see easy TS -> JS equivalent conversions:
        https://www.typescriptlang.org/play/

    Typescript Compiler (ts-node tool to compile and execute in one click):
        npm i -g typescript ts-node
        tsc --help

    Axios Dependency:
        npm install axios@0.27.2

    Fake JSON API:
        https://jsonplaceholder.typicode.com
    
    Initialize package.json (without installing any modules):
        npm init -y

    Install Parcel bundler:
        npm install -g parcel-bundler
        OR 
        npx parcel index.html
